# QNA_chatbot_using_ollama_openAI
The Enhanced Q&A Chatbot is a web-based application built using Streamlit and LangChain that enables users to interact with powerful Large Language Models (LLMs) for real-time question answering. The application supports both OpenAI models (GPT-4, GPT-4 Turbo, GPT-4o) and open-source models via Ollama (e.g., Mistral), providing flexibility in choosing between cloud-based and local AI solutions.

The chatbot features an intuitive user interface where users can input queries and receive intelligent, context-aware responses. It also allows customization of response behavior through adjustable parameters such as temperature and max tokens, enabling control over creativity and response length.

Additionally, the project integrates LangSmith tracing for monitoring and debugging LLM interactions, making it suitable for learning, experimentation, and real-world AI application development.

This project demonstrates practical implementation of:

LLM integration using LangChain

Prompt engineering techniques

Interactive UI development with Streamlit

API handling and environment configuration

Overall, it serves as a foundational project for building advanced AI-powered applications like virtual assistants, customer support bots, and knowledge-based systems.

This project is a Streamlit-based Q&A Chatbot that uses LangChain and LLMs to answer user queries in real-time.
